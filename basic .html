      Target attributes in links

      The destination files that is associated with the link can open at Multiple Locations

      visit google for search target Blank
      visit google for search (Parent)
      visit google for search (self)
      visit google for search (top)

      _blank = the linked document will be open in new tab or window

      _parent = the linked document will be open in new same window

      _self = the linked document will be open in the Parent frame

      _top = the linked document will be open in the full body of the window

      By default target is set to _self
